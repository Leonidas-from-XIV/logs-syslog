create clock tcp kv ~hostname ip ~port ~framing () is Ok reporter or Error msg. The reporter sends log messages to ip, port via TLS. If the initial TLS connection to the remote_ip fails, an Error msg is returned instead. If the TLS connection fails, attempts are made to re-establish the TLS connection. The hostname is part of each syslog message. The port defaults to 6514, framing to prepend the length in decimal.
