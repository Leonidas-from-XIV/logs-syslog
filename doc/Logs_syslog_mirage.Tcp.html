create clock tcp ~hostname ip ~port () is Ok reporter or Error msg. The reporter sends log messages to ip, port via TCP. If the initial TCP connection to the remote_ip fails, an Error msg is returned instead. If the TCP connection fails, attempts are made to re-establish the TCP connection. The hostname is part of each syslog message. The default value of port is 514, the default behaviour of framing is to append a 0 byte.
